# TP3-TBO

## Instruções do Makefile
Para compilar normalmente:
```bash
    make
```
Para compilar no modo `dubug` (isso fará com que o programa imprima o tempo de execução quando rodar ele):
```bash
    make debug
```

Rodar o programa:
```bash
    make run
```

Rodar o programa com valgrind:
```bash
    make valgrind
```
